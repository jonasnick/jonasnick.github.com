Part 1: Blockchain

  • Glossary: https://bitcoin.org/en/developer-documentation

Setup

  • create new datadir "alpha_datadir"
    • insert bitcoin.conf
  • start daemon alpha daemone with alphad -datadir=./alpha_datadir
  • alias a='alpha-cli -datadir=./alpha_datadir'
  • This is a private altcoin network

Exercises

  1. What are the arguments of the getrawtransaction RPC?
  2. What is the current blockchain height?
  3. What is the block interval?
  4. When was the genesis block created?
  5. Connect to the faucet and receive coins.
  6. Make it rain (spend your coins).

Getting familiar

  • help
    • help <cmd>
  • getinfo
  • getblockhash <block index>
  • getblock <block hash>

Receive coins

  • getnewaddress
  • connect to faucet with netcat -vv 188.166.154.40 2062
    • send address
  • check getbalance

Sending coins

  • sendtoaddress <address> <value>
    • returns transaction id
  • inspect transaction with gettransaction <txid>

List your addresses

  • getaddressesbyaccount ""
