Part 4: Fork Sidechain

Setup

  • Create new datadir
    • insert bitcoin.conf
  • Create new git branch

Exercises

  1. Create own chain with 2MB block size.
  2. Sign blocks.
  3. Find bitcoin script
  4. Explain the genesis block's coinbase transaction.

Increasing the blocksize

  • primitives/block.h increase MAX_BLOCK_SIZE

Creating block key

  • getnewaddress
  • dumpprivkey <address>
    • save result
  • validateaddress <address>
    • save pubkey

Customizing the chain

  • in src/chainparams.cpp
    • set scriptDestination to 1-of-1 multisig

Viewing txs that are not yet in the chain

  • getrawmempool

Signing blocks

  • getblockcount
  • getnewblockhex
  • signblock <blockhex>
  • combineblocksigs <blockhex> <'["signature"]'>
  • submitblock <signed blockhex>
  • getblockcount
